# 多面体字打印 (Multi-Directional Chinese Character 3D Models)

![示例效果](images/{BDE87C18-12CA-4C96-AE9E-737FC37A8B9A}.png)

这是一个基于OpenSCAD的3D模型生成工具，可以创建具有多个方向汉字的立体模型。当从不同角度观看时，这些模型会显示不同的汉字，是一种独特的创意设计工具，特别适合3D打印制作个性化礼品。

## 功能特点

- **多角度文字显示**：支持从两个、三个甚至更多方向显示不同汉字
- **参数化设计**：完全可定制的字体、大小、壁厚等参数
- **字体优化**：内置笔画连接增强算法，解决中文字体3D打印分离问题
- **多种模式**：支持完整模型或仅交合部分的渲染模式
- **底座支持**：灵活的底座设计选项，便于模型展示
- **自动优化**：提供角度自动优化功能，确保最佳显示效果
- **性能优化**：提供不同预览质量级别，加快设计过程

## 快速开始

### 安装要求

- [OpenSCAD](https://www.openscad.org/) (最新版本)
- 系统安装有适合3D打印的中文字体（推荐：华文楷体、华文中宋）
- 3D打印机（或3D打印服务）用于打印最终模型

### 使用步骤

1. 克隆或下载本仓库
2. 在OpenSCAD中打开主目录下的任意模型文件（如 `双面单字.scad`）
3. 修改参数设置您想要的文字和属性：
   ```openscad
   front_text = "福";    // 正面汉字
   back_text = "禄";     // 侧面汉字
   font = "华文楷体";     // 字体
   char_size = 20;       // 字体大小
   ```
4. 按F5预览，F6完整渲染
5. 导出STL文件（`File > Export > Export as STL`）
6. 使用切片软件准备3D打印

### 参数调整界面

在OpenSCAD中，您可以使用图形界面轻松调整模型参数：

![参数调整界面](images/{BDE87C18-12CA-4C96-AE9E-737FC37A8B9A}.png)

*参数界面说明：*
- **性能设置**：调整预览质量以平衡渲染速度和精度
- **参照面设置**：控制辅助面的显示方式，帮助定位和调整
- **字体设置**：设置各个面的文字内容、字体样式和大小
- **字符旋转设置**：调整字符在平面上的旋转角度
- **旋转与位置**：精确控制各个字面的X、Y、Z轴旋转角度
- **模型参数**：调整大小、壁厚、笔画连接等物理属性

您可以实时拖动滑块或输入具体数值，按F5查看效果。设置完成后，按F6进行完整渲染，然后导出STL文件。

## 模型类型

### 1. 单字模型系列

- **双面单字.scad**：在立方体的两个相邻面显示不同的单个汉字
- **三面单字.scad**：在立方体的三个相邻面显示不同的单个汉字
- **四面单字.scad**：在立方体的四个面显示不同的单个汉字

### 2. 多字阵列系列

- **双面横列.scad**：支持在两个面上显示任意长度的横排文本
- **双面纵列.scad**：支持在两个面上显示任意长度的竖排文本
- **三面多字阵列.scad**：支持在三个面上显示任意长度的文本，可选横排或竖排
- **四面多字阵列.scad**：支持在四个面上显示任意长度的文本，可选横排或竖排

## 示例参数

以下是一些预设参数组合的建议，可以直接在模型文件中使用：

- **福禄双全**（新年礼品）：

  ```openscad

  front_text = "福"; 
  back_text = "禄"; 
  font = "华文楷体";
  ```
- **福禄寿**（传统祝福）：

  ```openscad

  text1 = "福"; 
  text2 = "禄"; 
  text3 = "寿";
  font = "华文楷体";
  ```
- **新年快乐恭喜发财**（春节祝福）：

  back_text_direction = 1;  // 横排

## 使用说明

### 参数详解

1. **性能设置**
   - `preview_quality`：预览质量（0=快速预览，1=标准预览，2=高精度预览）
   
2. **参照面设置**
   - `show_reference_faces`：是否显示参照面
   - `face_identification`：面标识模式（0=无标识，1=显示面编号，2=显示面颜色）
   - `face_opacity`：参照面透明度（0-1）

3. **字体设置**
   - `front_text`/`text1`/`text2`等：各面显示的文字内容
   - `font`：字体名称
   - `char_size`：字体大小
   - `char_spacing`：字间距（多字阵列模型）

4. **阵列设置**（多字阵列模型）
   - `front_text_direction`：排列方向（0=竖排，1=横排）
   - `text1_direction`等：各面的排列方向

5. **模型参数**
   - `extrude_depth`：挤出深度
   - `wall_thickness`：壁厚
   - `stroke_offset`：笔画连接偏移量
   - `show_mode`：显示模式（0=交合部分，1=完整模型）

## 高级技巧

1. **笔画连接优化**：调整 `stroke_offset`参数（0.3-0.8）增强字体连接性，防止打印时笔画分离
2. **调整字符旋转**：使用`front_char_rotation`等参数调整字符在平面上的旋转角度，获得最佳视觉效果
3. **面角度调整**：通过调整X、Y、Z轴旋转角度，可以精确控制各个字面的方向和位置
4. **预览性能优化**：设计过程中使用`preview_quality=0`加快预览速度，最终渲染前设为更高值

## 问题排查

1. **问题**: 字体笔画分离，打印后脱落  
   **解决**: 增加 `stroke_offset`参数或选择更适合3D打印的连体字体

2. **问题**: 模型渲染很慢  
   **解决**: 降低 `preview_quality`值以加速预览，最终渲染前再调高

3. **问题**: 字符被截断或显示不完整  
   **解决**: 增加 `model_buffer`参数或调整 `base_cube_size`

4. **问题**: 找不到合适的字体  
   **解决**: 使用项目提供的字体查找工具（查看系统字体.bat），或参考字体指南
